# OSR (Open Source Recorder)
![alt text](./src/gambar.png "Title")

Project ini menggunakan Electron dan Typescript. Porject ini memiliki kelebihan dan kekurangan

## Kelebihan
1. Bisa menggunakan keybind untuk mulai atau berhenti recorder
2. Bisa mengganti Save Directory
3. Bisa mendapatkan audio desktop

## Kekurangan
1. Bisa mendapatkan audio microphone, tetapi tidak bisa digabungkan ke video.
